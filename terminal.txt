===
Installasi Library Pendukung
===
1. Installasi Vue Router
    > npm install vue-router@4.1.6
2. Installasi Vuex
    > npm install vuex@4.0.2 --save
3. Installasi Axios
    > npm install axios@1.3.4
4. Installasi Vue Content Loader
    > npm install vue-content-loader@2.0.1
5. Installasi VueperSlides
    > npm install vueperslides@3.5.1
6. Installasi Vue Toastification
    > npm install --save vue-toastification@2.0.0-rc.5
